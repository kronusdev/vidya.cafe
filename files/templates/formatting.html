Formatting

On {{"SITE_NAME" | app_config}}, you can use Markdown formatting.

Inline formatting

  • Italics: *hello* -> hello
  • Bold: **hello** -> hello
  • Code: `hello` -> hello
  • Strikethrough: ~~hello~~ -> hello
  • Links: [{{"SITE_NAME" | app_config}}]({{request.host_url}}) -> {{"SITE_NAME" | app_config}}

Block formatting

Markdown tags format an entire paragraph of text at a time.

  • Blockquote: >hello
  • Headers: # hello (headers go from # -> h1 to ##### -> h5)
    • Ordered List: 
    	1. hello
	2. hello
  • Unordered List: 
    	- hello
	- hello
  • Code Block: 
    	```
	hello
	hello
	```
  • Spoilers: <s>hello</s> ->
    hello

Custom Formatting

We also have some custom hooks for mentioning users, subreddits and polls. Note that these only work if the mentioned user or subreddit actually exists.

  • Username Mention: @Kronus -> @Kronus
  • Subreddit Mention: r/gaming -> r/gaming
  • Redditor Mention: u/spez -> u/spez
  • Polls: 
    	$$OPTION ONE$$
	$$OPTION TWO$$
    They look like in this post
